# Hrithik N L | Portfolio

Computer Science graduate and aspiring Full Stack Developer based in India. I enjoy building web applications using React, Node.js, and MongoDB.

## Features

- **Neo-Brutalist Design**: High-contrast, bold, and modern UI.
- **Project Showcase**: Detailed cards for my latest work.
- **About Me**: Categorized skills and professional bio.
- **Contact Form**: Integrated with Web3Forms for direct email communication.
- **Resume Access**: Easily viewable and downloadable resume.

## Technologies Used

- Vite
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Sonner (Toast notifications)

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/hrithiknl17/MyPortfolio.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file and add:
   ```env
   VITE_WEB3FORMS_ACCESS_KEY=your_key_here
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
